Markdown总结

  • summary

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。

Markdown 编写的文档可以导出 HTML 、Word、图像、PDF、Epub 等多种格式的文档。

Markdown 编写的文档后缀为 .md, .markdown。

标题

使用 # 号可表示 1-6 级标题，一级标题对应一个 # 号，二级标题对应两个 # 号，以此类推

段落格式

段落

Markdown 段落没有特殊的格式，直接编写文字就好，段落的换行是使用两个以上空格加上回车。

也可以在段落后面使用一个空行来表示重新开始一个段落。

字体

斜体：* 或 _ ；如*文本*,_文本_

粗体：** 或 __

粗斜体：*** 或 ___

分割线

可以在一行中用三个以上的星号、减号、底线来建立一个分隔线，行内不能有其他东西。也可以在星号或是减号中间插入空格。如：

删除线

如果段落上的文字要添加删除线，只需要在文字的两端加上两个波浪线 ~ 即可，如`文本~~`

下划线

下划线可以通过 HTML 的 <u> 标签来实现，如<u>文本</u>

脚注

脚注是对文本的补充说明。Markdown 脚注的格式如下:[^文本]

列表

无序列表

无序列表使用星号()、加号(+)或是减号(-)作为列表标记，这些标记后面要添加一个空格，然后再填写内容，如` 文本`

有序列表

列表嵌套

列表嵌套只需在子列表中的选项前面添加四个空格即可

区块

区块引用是在段落开头使用 > 符号 ，然后后面紧跟一个空格符号：

效果如下：

区块嵌套

一个 > 符号是最外层，两个 > 符号是第一层嵌套，以此类推

与列表嵌套方法类似

代码

行代码：用反引号，如：文本

代码区块：

代码区块使用 4 个空格或者一个制表符（Tab 键）
你也可以用 ``` 包裹一段代码，并指定一种语言（也可以不指定）

链接

[链接名称](链接地址) 或者 <链接地址>

图片

![alt 属性文本](图片地址)
![alt 属性文本](图片地址 "可选标题")

也可以使用普通的 <img> 标签

表格

表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行，如：

|  表头   | 表头  |
|  ----  | ----  |
| 单元格  | 单元格 |
| 单元格  | 单元格 |

对齐方式

-: 设置内容和标题栏居右对齐。
:- 设置内容和标题栏居左对齐。
:-: 设置内容和标题栏居中对齐
如下： 

| 左对齐 | 右对齐 | 居中对齐 |
| :-----| ----: | :----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |

转义

Markdown 使用了很多特殊符号来表示特定的意义，如果需要显示特定的符号则需要使用转义字符，Markdown 使用反斜杠转义特殊字符
